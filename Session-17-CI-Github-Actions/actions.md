# Actions in Github Actions

- action is samllest reusable unit in github action workflow
- its like a function or module that performs a specific task
- like:
    - checkout code
    - setup nodejs/python environment
    - deploy application in aws
- we can cretae workflows by using this inbuilt actions 

## Types of actions

1. official Actions
    - provided by Github
    - like checkout code 

2. Community actions:
    - created and shared by other developers in Github Marketplace
    - setup node

3. Custom Actions
    - create your own actions
    - javascript actions
    - docker actions (run inside container)
