# Configure Email Integration with Jenkins

## Set up email notification

- click on setting icon (manage jenkins)
- click on system
- scroll down to last (email notification)
- smtp server : stmp.gmail.com
- click on advanced (check)
- check SMTP authentication:
    - add username: type emailid
    - password: add your app password generated by email

- check on SSL - port number: 465
- click on Test Configuration check mail sent successfully or not
- if verification done then let's create project

## Set up project

- create free style project
- add description
- build step - execute shell - type exit 1(this will fail your job)
- i wanted to fail my job so i can receive an email
- post build action - select email notification - click on both check boxes
- build job
- when build fails you will receive an email.
