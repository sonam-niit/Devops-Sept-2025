# Introduction to Python

## Download Python

- Direct in windows: [Download Link](https://www.python.org/downloads/)
- Just follow basic installation steps and finish
- python --version (python -V)
- pip --version (pip -V)

**Pip is the package manager which is used to install packages inside python**

*Download in Ubuntu*

- sudo apt update 
- sudo apt install python3 -y
- python3 --version (-V)
- sudo apt install python3-pip -y
- pip3 --version (-V)

*Download in Linux (Amazon Linux Instance)*

- sudo yum install python3 -y
- python3 --version
- sudo yum install python3-pip -y

*Download in Mac Os*

- brew update
- brew install python
- python3 --version
- brew install pipx
- pip3 --version (-V)

## Variables

- temp storage for storing values
- value can be anything like number, text, condition check (true or false)

## Assignment

- to assign a value to variable we use assignment operator
- = operator we are using.

*rules for naming variables*

- variables name must start with letters or _
- it can include digit but not in start
- valid names: num1, hello_world, _user, age
- not valid names: x. , 1number, user name (space not allowed)
