# Implement Cypress Testing

- install Cypress
- initialize Project
- Write Test Case
- run test inside cypress runner
- required NodeJS to setup cypress

```bash
npm -v
node -v
# create folder cypress-demo
# move to that folder
# create package.json file
npm init -y
# install cypress
npm i cypress --save-dev
# launch cypress using command
npx cypress open
```
- when you run above command it will open cypress screen
- select E2E testing
- click on create new spec with default code

- you can check in your VS Code under the cypress folder e2e
- folder generated below that spec you can see
- edit some code and check in browser
- you can see each test execution step by step

- try to add scafolded test case
- you can see all inbuild test of cypress
