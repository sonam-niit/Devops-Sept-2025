# CD Orchestration Flow (Not Manual)

- Code commit on Github

- CI Build + Unit Test

- Artifact Stored (JAR / dist / Docker Image)

- Deploy to Staging 

- Smoke Test ( test basic functionality)

- Deploy to Production

- Traffic Shift (Rolling / Canary/ blue-green )

- Monitoring

- Auto Rollback (if failed)

*CD = Automation + Orchestration*
