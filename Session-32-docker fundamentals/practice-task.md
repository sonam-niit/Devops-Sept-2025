# Practice From Docker

1. Create one simple portfolio website using HTML/CSS
    - use AI tool to create and design your website
    - once the website is ready
    - create docker file
    - use nginx as base image
    - copy your code to default location of nginx
    - expose on port 80

2. Build Image
3. Run image in container named portfolio on external port 9090
4. take screenshot of browser output
    localhost:9090

5. portfolio must be accessed here.
